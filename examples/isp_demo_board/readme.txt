Both files are to be used with the "Altera ISP Demo Board". 

9320.jam will program an EPM9320RC208, on the Altera ISP Demo Board
with an LED game. The game is played by moving the small triangle, at the
end of the LED matrix, to avoid the falling dots. The triangle is moved
by pressing "SW1" and "SW2" on the Demo Board. The game is reset by 
pressing both switches down at the same time.

7128S.jam will program an EPM7182SQC100, on the Altera ISP Demo Board
with a simple LED pattern. When pressing the "SW1" key, the top row of
LED's will light up. When SW1 is released the whole LED matrix will light
up.
